## 应用简介:
口袋是一个工具与资讯聚合类应用，在[七磅](https://github.com/1anc3r/SevenPounds)的基础上加入很多以往项目中的代码，算是对我的Android学习之旅的阶段性总结。口袋一方面可以作为工具发短信打电话、管理各类型文件、查询天气、简单计算、翻译单词、摩斯电码、二维码扫描与生成，另一方面可以作为获取资讯的方式，提供了文章、趣闻、段子、图书、音乐、电影、妹子图、漫画、视频、游戏、编程方面的资讯，提供夜间模式和主题换色。

## 功能介绍:
    
### 工具

1. 电话、通讯录、信息 : 
    
    电话 : 列出手机上的通话记录
    
    通讯录 : 列出手机上的联系人
    
    信息 : 列出手机上的短信记录
    
    
2. 图片、音乐、视频、文档、应用、存储 : 
    
    列出手机上的各种类型的文件
    
    支持打开、删除、赋值、剪切
    
    
3. 日历 : 
    
    一个普通的日历
    
    需要手动添加事项
    
    
4. to-do : 
    
    一个普通的to-do
    
    各方面都很普通，不过是Google Sample改的
    
    
5. 天气 : 
    
    天气信息 : 提供全国各城市的天气信息
    
    城市选择 : 然而并不能定位
    
    — 数据来源 : [中央天气](tj.nineton.cn/Heart/index)
    
    
6. 翻译 : 
    
    — 数据来源 : [必应词典](cn.bing.com/dict/)
    
    
7. 摩斯电码 : 
    
    在字符串和摩斯电码之间任意转换
    
    长按粘贴, 双击复制
    
    
8. 计算器 : 
    
    这是一个支持多项式运算的计算器, 由栈实现
    
    
9. 二维码 : 
    
    扫描 : 调用后置摄像头扫描二维码
    
    识别 : 识别本机图片中的二维码
    
    — 使用开源库 : (github.com/scola/Qart)
    
    
### 资讯

1. 文章 : 
    
    每日一文 : 每天一篇精选优质短篇
    
    随机文章 : 点击刷新按钮随机读文章
    
    — 数据来源 : [每日一文](meiriyiwen.com)
    
    
2. 趣闻 : 
    
    每日 : 知乎日报的每日信息
    
    热门 : 知乎日报的热门信息
    
    分类 : 包括动漫、游戏、财经、电影、音乐、互联网安全等日报
    
    — 数据来源 : [知乎日报](news-at.zhihu.com/api)
    
    
3. 段子 : 
    
    段子 : 内涵段子的热辣段子
    
    图片 : 内涵段子的爆笑图片
    
    — 数据来源 : [内涵段子](neihanshequ.com)
    
    
4. 图片 :  
    
    妹子 : 好看的妹子图
    
    美景 : 好看的风景照
    
    — 数据来源 : [妹子 Gank.io](gank.io)、[美景 Pexels](www.pexels.com)
    
    
5. 图书 : 
    
    书评 : 豆瓣读书的**最受欢迎**书评
    
    书榜 : 爬取呈现豆瓣图书**TOP250**
    
    搜索 : 点击右上角的搜索按钮，搜索你想了解的图书信息
    
    — 数据来源 : [豆瓣读书](book.douban.com)
    
    
6. 音乐 : 
    
    乐评 : 豆瓣音乐的**最受欢迎**乐评
    
    乐榜 : 爬取呈现豆瓣音乐**TOP250**
    
    搜索 : 点击右上角的搜索按钮，搜索你想了解的音乐信息
    
    — 数据来源 : [豆瓣音乐](music.douban.com)
    
    
7. 电影 : 

    影评 : 豆瓣电影的**最受欢迎**影评
    
    影榜 : 爬取呈现豆瓣电影**TOP250**
    
    搜索 : 点击右上角的搜索按钮，搜索你想了解的电影信息
    
    — 数据来源 : [豆瓣电影](movie.douban.com)
    
    
8. 视频 : 
    
    分类 : 来自B站各分区**排行榜**前十
    
    — 数据来源 : [BiliBili](api.bilibili.com)
    
    
9. 游戏 : 
    
    精选 : 精选各大平台热门游戏

    优惠、热销、新品、即将推出
    
    — 数据来源 : [Steam](store.steampowered.com) 
    
    
10. 小说 : 
    
    排行 : 小说**排行榜**
    
    分类 : 小说各分区
    
    搜索 : 点击右上角的搜索按钮搜索你想看的小说
    
    — 数据来源 : [追书神器](www.zhuishushenqi.com)
    
    
11. 漫画 : 

    推荐 : 推荐好看的漫画
    
    排行 : 漫画**排行榜**
    
    分类 : 来自有妖气各分区**排行榜**
    
    — 数据来源 : [有妖气漫画](www.u17.com)
    
    
12. 编程 : 

    个人 : GitHub上Star**最多**的个人
    
    组织 : GitHub上Star**最多**的组织
    
    项目 : GitHub上Star**最多**的项目
    
    趋势 : GitHub上今日**最热**的项目
    
    — 数据来源 : [GithubRanking](github-ranking.com)

## 技术要点

项目采用 MVP 架构，Okhttp 和 Volley 加载网络数据，JSoup 解析 Json 数据， Glide 加载图片资源，SQLite 读写本机数据；使用 RecyclerView 加载多种布局，支持下拉刷 新上拉加载；应用支持主题换色、夜间模式、亮度调节。通过此项目掌握了 MVP 设计思想， 掌握了多种数据处理方式，学习了各种Android的性能优化技巧。
